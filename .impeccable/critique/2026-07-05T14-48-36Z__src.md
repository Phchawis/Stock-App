## Anti-Patterns Verdict

**Verdict**: Theme Fragmentation & Layout Sameness.

- **LLM assessment**: The visual theme is split between the dark teal-green main layout ("Emerald Ocean") and the light blue-grey modals (#e0ecf0 background). This layout and styling disconnect represents a clear tell of prompt-fragmentation.
- **Deterministic scan**: The CLI detector returned 97 raw findings. After filtering 56 false positives (mostly print stylesheets and dark theme variable initializations), 41 active design-system deviations remain.
  - **Main.jsx (15 findings)**: Out-of-system color values on buttons (using Tailwind `#10B981` instead of primary `#0E9587`).
  - **Sidebar.jsx (11 findings)**: Out-of-system sky blue hover shadow `#1A93B3`.
  - **DetailDrawer.jsx (8 findings)**: Out-of-system green/amber button highlights.
  - **Dashboard.jsx (5 findings)**: Hardcoded `#2E9E63` and `#1387A6` in charts.
  - **Alerts.jsx & IssueModal.jsx (2 findings)**: Hardcoded colors like `#1387A6` and `#38B673`.

## Overall Impression
The app offers a clean layout and beautiful custom Thai lab terminology, but is held back by minor color drift and visual contrast issues. Aligning the color systems across pages is the single biggest design opportunity.

---

## What's Working
1. **Sensory Scan Feedback**: The haptic vibration and screen green flashes when scanning QR codes are excellent and highly functional.
2. **Mobile Layout Collapsing**: The screens correctly drop down into single-column card grids on narrow screen viewports.

---

## Priority Issues

### [P0] Broken Category Filter in Alerts Screen
- **Why it matters**: Alerts page hardcoded categories `['HMS', 'ADV']` do not match the real database categories.
- **Fix**: Dynamically extract category values from the active database reagents.
- **Suggested command**: `/impeccable polish`

### [P1] Visual Palette Disconnect (Dark Teal vs Light Modals)
- **Why it matters**: Main pages use a dark teal-green theme, but modals override variables to use a light blue-grey palette, causing visual clash.
- **Fix**: Remove the light theme overrides to let modals inherit the primary "Emerald Ocean" dark styling.
- **Suggested command**: `/impeccable colorize`

### [P1] Keyboard-Inaccessible Custom Selects
- **Why it matters**: Custom SearchableSelect input blocks keyboard users and screen readers.
- **Fix**: Support ArrowUp/ArrowDown/Escape/Enter and add proper ARIA combobox tags.
- **Suggested command**: `/impeccable clarify`
